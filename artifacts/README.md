## открытые эфиры

перед стартом лабы программа собирается в прямом эфире (три открытые рабочие сессии). практика в моменте: собрать плагин/скилл вживую, спросить модель «как сделать X», собрать контекст о себе через интервью с моделью, обмен контекстом между агентами. записи и материалы каждой сессии — на [сайте лабы](https://s26.aimindset.org/).

- **эфир #1 — почему ai-инструменты не сделали рабочую систему легче.** внимание и контекст: зачем нужны фреймворки продуктивности, где польза и где хайп, что реально даёт агентный стек; практика — сбор личного контекста через интервью с моделью.
- **эфир #2 — личная система: контекст, shortcuts, автоматизация и ритм.**
- **эфир #3 — агентный цикл и человеческая приёмка (a2a / personal corp).**

*артефакты конкретных сессий (конспекты, один практический шаг) добавляются после каждого эфира в обезличенном виде.*
