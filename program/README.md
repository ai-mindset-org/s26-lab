## дуга: шесть элементов

порядок держит логику сборки — каждый следующий элемент опирается на предыдущий.

| # | элемент | что происходит |
|---|---------|----------------|
| 1 | **зачем** | на открытии участник вслух называет, что соберёт за месяц. публичное обещание закрепляет цель. |
| 2 | **контекст** | аудит жизни, инвентаризация поверхностей, чистка сохранёнок. агент получает профиль человека. |
| 3 | **состояние** | поток встаёт на собранный контекст: без ясной цели среда не удержит фокус. |
| 4 | **агенты** | поверх контекста ложится агентный цикл — недельный план и ретро вместе с агентом. |
| 5 | **люди** | участники показывают работу друг другу, сверяют инструменты, растят карту подходов. |
| 6 | **форма** | месяц сходится в финальный артефакт и разбор «до и после» на demo day. |

## недели

| фаза | фокус | артефакт |
|------|-------|----------|
| **pre-lab** | онбординг, сбор контекста | входной документ + проверка доступов |
| **W1** | зачем · контекст · состояние | карта жизни, context layer, первый агентный цикл v0.1 |
| **W2** | агенты · люди | agent loop ×2 + карта делегирования |
| **W3** | форма · ship | рабочий инструмент / карта / протокол |
| **demo day** | до / после | публичный показ + план внедрения |

## ритм

- **лекция** — ядро недели, задаёт рамку;
- **коворкинг** — совместная работа над своим процессом;
- **гостевой слот** — практик приносит один законченный кейс;
- **office hours** — разбор застреваний;
- **хакатон** — субботний рабочий интенсив, где хватает времени довести до результата;
- **demo day** — публичный показ работающих инструментов.

устойчивый поток держится на осцилляции: работа чередуется с восстановлением. героизм с ночными переработками ломает навык.

## карта подходов

сквозной артефакт месяца. две оси:

```
                    структура
                        │
          порядок ──────┼────── бардак
                        │
                     интерес
```

проводники ставят первые точки (разные способы собирать личную систему), участники добавляют свои. карта служит языком рефлексии на реальной задаче: где твой подход и куда ты хочешь сместиться.

## что участник выносит

один дорогой рабочий процесс, проведённый через весь месяц, и устойчивая конфигурация **человек + агенты + люди вокруг**:

- контекст-слой для одного ключевого процесса;
- правило человеческого вмешательства (где заканчивается делегирование);
- проверенный агентный цикл с видимым следом решений;
- работающий инструмент, карту или протокол к demo day.
