# STYLE · визуальная грамматика shaper

все артефакты этого репозитория (диаграммы, схемы, карты) оформляются на одном языке — **shaper**. это делает базу знаний читаемой как единую систему: любой рисунок узнаётся с первого взгляда и стыкуется с соседними.

shaper это не «ретро-терминал», а **современный технический чертёж / патентная иллюстрация**: аскетичный чёрно-белый язык, где смысл несёт структура, а не цвет.

## токены

канонические значения лежат в [`assets/shaper.css`](assets/shaper.css). коротко:

```
поверхность   bg #ffffff (бумажно-белый) · ink #000000 (чистый чёрный)
серые (три)   soft #d9d9d9 · hair #ececec · muted #8a8a8a
шрифт         JetBrains Mono (моно), больше ничего
геометрия     рамка всегда 1px · углы прямые · без теней и градиентов
акцент        flow-жёлтый #e9b400 — только точкой, дозой, никогда как фон текста
```

## правила

1. **строчные буквы** везде, кроме CAPS микро-подписей (`letter-spacing: .18em`).
2. **нумерованный блок** — основная единица композиции: рамка 1px + чёрный кружок с номером в левом верхнем углу.
3. **инверсия = активное состояние.** выделенное, текущее, изменённое = `background: ink; color: bg`. никогда не цветом, не жирным, не подсветкой.
4. **связи — 1px линии со стрелкой-треугольником**, а не текстовые стрелки и не иконки.
5. **контраст через размер и прозрачность**, не через толщину шрифта.
6. **никаких** цветов как смысла, sans-serif, Title Case, border-radius (кроме кружков-номеров), эмодзи в заголовках, иконок-наборов, Tailwind.

> проверка на «это не shaper»: если без двух и более цветов смысл теряется — язык выбран неверно.

## как сделать новый артефакт

1. скопируй скелет из любого файла в [`assets/`](assets/) (например [`s26-lab-map.html`](assets/s26-lab-map.html)).
2. замени содержимое `<main>`, оставь header/footer как есть.
3. добавляй нумерованные блоки, помечай текущее состояние инверсией.
4. рендер в png (retina 2x) для вставки в markdown; html остаётся исходником.

примеры в репозитории: [карта сборки лабы](assets/s26-lab-map.png) · [анатомия скилла /research](assets/research-skill-anatomy.png).
